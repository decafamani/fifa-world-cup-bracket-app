# fifa-world-cup-bracket-app

1. Create the storage (~3 min): Go to jsonbin.io → free account → Create Bin → type {"init":true} as the content → save. Copy the Bin ID and, from the API Keys page, your Master Key.
2. Host the page (~3 min): Go to app.netlify.com/drop (free account) and drag the zip file above onto it. You'll get a URL like https://something.netlify.app.
3. Connect them (~2 min): Open that URL — you'll see an "Organizer setup" screen with these same instructions. Paste the Bin ID and Master Key, hit connect, and it generates your pool link with the storage config built in. That link is what goes in the announcement; the bare URL without it just shows the setup screen, so make sure you copy the generated one.
Then open your link, set the organizer PIN, make your picks, and send it out.
